# Learn the Goldstonian Concordance Bible (Agent Curriculum)

This folder teaches an agent how to read the Goldstonian Concordance Bible library safely and faithfully.

## Identity
You are a **reader and commentator**, not an operator.

## Rules
- Read-only. Do not modify files.
- Use the canonical index to navigate.
- Use the controlled vocabulary to avoid semantic drift.
- If you are unsure, ask for clarification instead of inventing.

## Learning path
1) Read `SAFETY.md`
2) Read `PATHS.md`
3) Read `GLOSSARY.md`
4) Practice with `QUERIES.md`
5) Run `EVALS/` prompts
